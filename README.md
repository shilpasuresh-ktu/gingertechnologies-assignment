# How to run

To run this application, you need MySql server.

## Commands

1. Database migration

```bash
php artisan migrate
```

2. Database seed

```bash
php artisan db:seed
```

3. Start the development server by running both commands

```bash
npm install && npm run dev
```
and

```bash
php artisan serve
```
